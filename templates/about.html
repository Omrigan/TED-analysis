TED talks topics analyser

Okay, well, I am a python developer, my name is Oleg Vasilev. This is a data analysis experiment made for GoTo Hack.
The main idea is try to determine top-20 topics of TED talks.

Algorithm

The problem contains three unrelated parts:

Parse data

The easiest part. Just downloading srt files, parsing data from web page and saving it to MongoDB collection

Model building

I am using genism python library to build the LDA model ans save it to my_model file

Talks analysis

Here we are processing each talk to recognize which topics are mentioned in the subtitles, and its weight.
For each topic then we find which part of total speaks, views and discussions it consist.
All source code is hosted on GitHub

Used technologies

  • Python
  • Genism for linguistic analysis
  • Flask for data presentation
  • Pyquery for parsing
  • and so on...
  • MongoDB for storage
  • Digital Ocean for hosting
